# MIKCONFIG
MIKCONFIG is a configurator tool for mikrotik devices.
MIKCONFIG allows for the configuration of a mikrotik device
Given the following properties:
* **ROUTER**
  * **DEVICE NAME**
  * **FAILOVER**
    * WAN PORTS
  * **INTERNAL NETWORK**
    * INTERNAL GATEWAY
    * INTERNAL IP POOL
    * INTERNAL VLAN NAME / ID
    * INTERNAL WIFI NAME / PASSWORD
  * **GUEST NETWORK**
    * GUEST GATEWAY
    * GUEST IP POOL
    * GUEST VLAN NAME / ID
    * GUEST WIFI NAME / PASSWORD
  * **DNS SERVERS**
* **OPEN VPN SERVER**
  * SERVER PORT
  * VPN SERVER IP
  * VPN NETWORK
  * VPN POOL
  * VALIDITY
  * Nº OF CLIENTS
  * USERNAME AND PASSWORD PER CLIENT
* **SSTP SERVER** 
  * SERVER PORT
  * VPN SERVER IP
  * VPN NETWORK
  * VPN POOL
  * VALIDITY
  * Nº OF CLIENTS
  * USERNAME AND PASSWORD PER CLIENT
